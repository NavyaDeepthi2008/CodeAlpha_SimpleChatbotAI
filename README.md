```md
# Basic Chatbot
```


## Description
This is a simple rule-based chatbot developed using Python.  
The chatbot responds to predefined user inputs and interacts through the console.

## Features
- Simple text-based conversation
- Predefined responses
- Continuous chat loop
- Exit using "bye"
- Easy to understand logic

## Technologies Used
- Python

## Concepts Used
- If-elif statements
- Functions
- Loops
- Input/Output

## How to Run
1. Install Python on your system.
2. Open terminal or command prompt.
3. Run the following command:

```bash
python chatbot.py
```
